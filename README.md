# Physics-Enhanced Orthogonal Sensing for Self-Supervised Anomaly Detection in Rolling Mills

This repository contains the standalone data preprocessing module for the anomaly detection framework presented in the paper. It demonstrates how to transform raw 4-channel sensor data (Pressure and Vibration) into 16x16 Cross-Spectral Density (CSD) matrices.

## Key Components

1. **Virtual Channel Expansion (WPD)**: 
   - Uses Wavelet Packet Decomposition (WPD) with depth $L=3$.
   - Selects the top $K=3$ frequency bands based on energy and entropy.
   - Expands the 4 physical channels (P1, V1, P2, V2) into 16 virtual channels.

2. **CSD Matrix Construction (Welch's Method)**:
   - Computes the Cross-Spectral Density matrix for the 16 virtual channels using Welch's method.
   - Outputs a $32 \times 32$ real-valued matrix (representing the $16 \times 16$ complex matrix).

## Files

- `preprocess.py`: The standalone Python script containing the `VirtualChannelExpander` and `CSDMatrixBuilder` classes.
- `sample_data.csv`: A small, anonymized sample of raw sensor data (1000 rows).
- `sample_csd.npy`: The resulting CSD matrix generated by running the preprocessing script.

## How to Run

1. Ensure you have the required dependencies installed:
   ```bash
   pip install numpy pandas scipy PyWavelets
   ```

2. Run the preprocessing script:
   ```bash
   python preprocess.py
   ```

3. The script will read `sample_data.csv`, perform the WPD expansion and CSD computation, and save the result as `sample_csd.npy`.
